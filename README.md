# 내벨업 테스트룸 (Level Up Test Room)

프로그램별 맞춤 학습 일정과 과제를 제공하는 영어 시험 연습 플랫폼입니다. Reading, Listening, Speaking, Writing 4개 영역의 체계적인 학습을 지원합니다.

**🔐 수강생 인증 시스템**: Google Sheets를 통해 등록된 수강생만 접근할 수 있으며, 프로그램별로 맞춤 학습 일정이 제공됩니다.

## 🚀 빠른 시작

### 1단계: Google Sheets 수강생 명단 만들기

1. **Google Sheets에서 새 스프레드시트 생성**

2. **첫 번째 시트: 수강생 명단**
   - 첫 번째 행에 헤더 입력:
   ```
   | 이름    | 휴대폰번호   | 프로그램 |
   |---------|-------------|---------|
   | 홍길동  | 01012345678 | 일반    |
   | 김철수  | 01087654321 | 급행    |
   | 이영희  | 01055556666 | 일반    |
   ```
   - **프로그램**: `일반` 또는 `급행` (4주 vs 8주 과정)

3. **스프레드시트 공개 설정**:
   - 우측 상단 "공유" 버튼 클릭
   - "액세스 권한" 섹션에서 "제한됨" → "링크가 있는 모든 사용자" 선택
   - 권한: "뷰어" (읽기 전용)
   - "완료" 클릭

4. **Spreadsheet ID 추출**:
   - 브라우저 주소창의 URL 확인
   - 예: `https://docs.google.com/spreadsheets/d/1ABC...XYZ/edit#gid=0`
   - `/d/` 다음부터 `/edit` 전까지가 Spreadsheet ID
   - 예: `1ABC...XYZ`

### 2단계: 프로젝트 설정

**`js/auth.js` 파일을 열고 Spreadsheet ID 입력**:

```javascript
const SHEET_CONFIG = {
    spreadsheetId: '1vyi3LV5bZNQ0dxsZOjpde94BD8aTwtu-MvzCVgtxOIE',  // 실제 ID로 변경
    sheetGid: '0',
};
```

### 3단계: 테스트

1. `index.html` 파일을 브라우저에서 열기
2. Google Sheets에 등록한 이름과 휴대폰번호로 로그인
3. 프로그램에 맞는 학습 일정 확인
4. 원하는 Week와 요일 선택하여 과제 시작!

## 🎯 주요 기능

### ✅ 현재 구현된 기능

#### 0. **프로그램별 학습 시스템** 🎓
- **Standard 프로그램**: 8주 과정 (Week 1~8)
- **Fast 프로그램**: 4주 과정 (Week 1~4)
- 프로그램별 맞춤 학습 일정 자동 표시
- 주차별, 요일별 과제 관리
- **Week 2 화요일**: 내벨업보카 시험 (Standard: p.5-6 / Fast: p.5-8)

#### 1. **수강생 인증 시스템** 🔐
- Google Sheets 기반 수강생 명단 관리
- 이름 + 휴대폰번호 + 프로그램 정보 통합 관리
- 세션 유지 (새로고침 시에도 로그인 상태 유지)
- 로그아웃 기능
- 등록되지 않은 사용자 차단
- 데모 모드 지원

#### 2. **학습 일정 화면** 📅
- 프로그램별 주차 자동 표시 (일반 4주 / 급행 8주)
- 주차별 요일 선택 UI (일~금, 토요일 제외)
- 각 날짜별 과제 설명 미리보기
- 반응형 그리드 레이아웃

#### 3. **Reading 섹션** (54분)
- 3개의 지문과 총 10개의 문제
- 실제 토플 형식의 지문 독해
- 지문과 질문을 동시에 볼 수 있는 분할 화면
- 답안 선택 및 저장 기능
- 이전/다음 문제 탐색
- 실시간 타이머

#### 4. **Listening 섹션** (41분)
- 대화(Conversations)와 강의(Lectures) 포함
- 오디오 재생 인터페이스
- 메모 작성 가능
- 객관식 문제 답변
- 실시간 타이머

#### 5. **Writing 섹션** (50분)
- 2개의 라이팅 과제
  - Integrated Writing Task (20분)
  - Independent Writing Task (30분)
- 텍스트 편집 도구 (볼드, 이탤릭, 밑줄)
- 실시간 단어 수 카운터
- 자동 초안 저장
- 지문과 에디터 분할 화면

#### 6. **Speaking 섹션** (17분)
- 4개의 스피킹 과제
  - Independent Speaking Task
  - Integrated Speaking Tasks (3개)
- 준비 시간 타이머 (15-30초)
- 응답 시간 타이머 (45-60초)
- 마이크를 통한 음성 녹음 기능
- 메모 작성 공간
- 녹음 진행 상태 시각화

#### 8. **내벨업보카 시험** 📚 (페이지 기반 시스템)
- **Google Sheets 연동** 단어 시험 시스템
- **페이지 기반** 시험: 교재 페이지 번호로 자동 필터링
  - Standard 프로그램: Week 2 화요일 (p.5-6), Week 3 화요일 (p.9-10), Week 4 화요일 (p.13-14) 등
  - Fast 프로그램: Week 2 화요일 (p.5-8), Week 3 화요일 (p.9-12), Week 4 화요일 (p.13-16)
- 표제어(Headword)를 보고 **동의어(Synonyms) 입력**
- 동의어 개수에 따라 입력칸 자동 생성 (**최대 8개까지 지원**)
- 철자 정확도 100% 요구 (한 글자라도 틀리면 오답)
- 실시간 채점 및 결과 화면
- 정답/오답 비교 및 상세 피드백 제공
- 다시 시도하기 기능

**데이터 구조 (Google Sheets)**:
```
| page | headword  | synonym1   | synonym2     | synonym3    | ... | synonym8 |
|------|-----------|------------|--------------|-------------|-----|----------|
| 5    | coincide  | occur together | match    |             |     |          |
| 5    | collapse  | fall       | break down   |             |     |          |
| 6    | comprehensive | complete | thorough  | extensive   |     |          |
```

**페이지 기반 시스템**:
- 각 단어에 페이지 번호 할당
- 프로그램/주차에 따라 자동으로 해당 페이지만 로드
- Google Sheets에서 한 번만 데이터 입력하면 모든 프로그램에 자동 적용

**시험 흐름**:
1. Week 선택 → 화요일 클릭 → 자동으로 해당 페이지 시험 로드
2. 시험 소개 화면 → 주의사항 확인
3. 각 표제어에 대한 동의어 입력 (입력칸 수는 동의어 개수와 동일, 최대 8개)
4. 모든 문제 풀이 후 제출
5. 채점 화면: 정답률, 문제별 결과, 내가 쓴 답 vs 정답 비교

#### 9. **타이머 시스템**
- 각 섹션별 정확한 시간 제한
- 실시간 카운트다운
- 1분 미만 시 경고 색상 표시
- 시간 초과 시 자동 제출

#### 10. **결과 화면**
- 총점 및 섹션별 점수 표시 (120점 만점)
- Reading/Listening 자동 채점
- 결과 JSON 파일 다운로드
- 시험 재시작 옵션

#### 11. **반응형 디자인**
- 데스크톱, 태블릿, 모바일 지원
- 터치 친화적 인터페이스
- 적응형 레이아웃

## 📝 사용 방법

### 1. 로그인
1. Google Sheets에 등록된 이름과 휴대폰번호를 입력합니다
2. "인증하기" 버튼을 클릭합니다
3. 인증 성공 시 학습 일정 화면으로 자동 이동합니다
4. 프로그램 배지에서 본인의 프로그램 확인 가능 (일반/급행)

### 2. 학습 일정 선택
1. 화면에 표시된 주차 중에서 원하는 주차 선택
   - Fast 프로그램: Week 1~4
   - Standard 프로그램: Week 1~8
2. 해당 주차의 요일 버튼 클릭 (일~금, 토요일 휴무)
3. 과제 목록 화면에서 원하는 과제 선택
4. 자동으로 과제 화면으로 이동

**과제 타입별 동작**:
- **내벨업보카**: 해당 페이지 단어 시험 시작
- **입문서 정독**: PDF 링크 모달 팝업
- **리딩/리스닝 Module**: 해당 모듈 시작 (구현 예정)
- **라이팅/스피킹**: 해당 번호 과제 시작 (구현 예정)

### 3. 과제 수행

#### Reading 섹션
1. 왼쪽에 지문, 오른쪽에 문제가 표시됩니다
2. 답안을 클릭하여 선택합니다
3. "Next" 버튼으로 다음 문제로 이동합니다
4. "Previous" 버튼으로 이전 문제로 돌아갈 수 있습니다
5. 마지막 문제에서 "Submit Section"을 클릭하여 제출합니다

#### Listening 섹션
1. 오디오를 주의깊게 듣습니다 (메모 가능)
2. "Continue" 버튼을 클릭하여 문제로 이동합니다
3. 각 문제에 답변합니다
4. 모든 문제를 완료하면 다음 오디오로 이동합니다

### Speaking 섹션
1. 과제를 읽고 "Start Preparation"을 클릭합니다
2. 준비 시간 동안 메모를 작성합니다
3. 자동으로 녹음이 시작됩니다
4. 제한 시간 내에 응답합니다
5. 녹음이 자동으로 종료되거나 수동으로 중지할 수 있습니다

### Writing 섹션
1. 왼쪽에 프롬프트, 오른쪽에 에디터가 표시됩니다
2. 에세이를 작성합니다
3. 에세이를 작성합니다
4. 단어 수가 실시간으로 표시됩니다
5. "Save Draft"로 초안을 저장할 수 있습니다
6. "Next Task"로 다음 과제로 이동합니다

### 내벨업보카 시험 (Week 2 - 화요일) 📚
1. 시험 소개 화면에서 주의사항을 확인합니다

3. 각 표제어를 보고 모든 동의어를 입력합니다
   - 동의어 1개: 입력칸 1개
   - 동의어 2개: 입력칸 2개
   - 동의어 3개: 입력칸 3개
4. 모든 문제를 풀고 "제출하기" 버튼을 클릭합니다
5. 채점 결과 화면에서 다음을 확인합니다:
   - 총 정답률 (%)
   - 정답/오답 개수
   - 문제별 상세 결과 (내가 쓴 답 vs 정답)
6. "다시 시도하기" 또는 "학습 일정으로" 버튼 선택

## 🆘 자주 묻는 질문 (FAQ)

### Q: Google Sheets 설정이 어렵습니다
A: `js/auth.js`에서 `SHEET_CONFIG`를 설정하지 않으면 자동으로 데모 모드로 작동합니다. 테스트 후 천천히 설정하세요.

### Q: "수강생 목록을 불러오는데 실패했습니다" 에러가 나옵니다
A: Google Sheets의 공개 설정을 확인하세요. "링크가 있는 모든 사용자"로 설정되어 있어야 합니다.

### Q: 휴대폰번호 형식은 어떻게 입력하나요?
A: 하이픈(-) 없이 숫자만 입력합니다. 예: 01012345678

### Q: 시험 도중 새로고침하면 어떻게 되나요?
A: 로그인 상태는 유지되지만, 현재 진행 중인 시험 답안은 초기화됩니다.

### Q: 결과는 어떻게 저장되나요?
A: 현재는 브라우저에서만 표시되고 JSON 파일로 다운로드할 수 있습니다. 향후 Google Sheets에 자동 저장 기능이 추가될 예정입니다.

### Q: 내벨업보카 시험이 로드되지 않습니다
A: Google Sheets 공개 설정을 확인하세요:
1. 시트 우측 상단 "공유" 버튼 클릭
2. "일반 액세스" → **"링크가 있는 모든 사용자"** 선택
3. 권한: **"뷰어"** (읽기 전용)
4. "완료" 후 페이지 새로고침
